Sales Analysis using Python (Pandas)

This project focuses on performing sales data analysis using Python libraries such as Pandas, Matplotlib, and Seaborn. The analysis is carried out on a sample Superstore dataset using Jupyter Notebook (Anaconda environment).

🎯 Objective

To perform data cleaning, manipulation, and exploratory data analysis (EDA) using Pandas and extract meaningful business insights from sales data.

🛠️ Tools & Technologies Used
Python
Pandas
Matplotlib
Seaborn
Jupyter Notebook (Anaconda)
📌 Key Analysis Questions
Which products are the most sold?
What are the top 10 products based on sales?
Which products generate the highest profit?
Which category has the highest sales?
Which category is the most profitable?
What are the total sales values by category and sub-category?
Which products are most sold within each sub-category?
Which customer segments are the most profitable?
What is the impact of shipping modes on sales?
How do shipping modes affect category-wise sales?
Which market contributes the highest sales?
What are the top 10 countries by sales?
Which countries have the highest average shipping cost?
Who are the top 10 most profitable customers?
What are the total sales trends by year and month?

This project helps in understanding sales trends, customer behaviour, and profitability insights, which can support better business decision-making.
